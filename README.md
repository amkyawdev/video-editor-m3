# Video Editor M3

A powerful Android video editor application with FFmpeg integration.

## Developer

**AUNG MYO KYAW** (Developer)

## Features

- Video trimming and editing
- FFmpeg-powered video processing
- Export videos in various formats
- Project management with local database
- Modern Jetpack Compose UI

## Run Locally

**Prerequisites:** [Android Studio](https://developer.android.com/studio) | JDK 17+

1. Open Android Studio
2. Select **Open** and choose the directory containing this project
3. Allow Android Studio to sync Gradle files
4. Run the app on an emulator or physical device
